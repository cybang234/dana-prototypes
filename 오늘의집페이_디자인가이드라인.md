## 1. 환경 조건

### 1-1. 화면 사이즈

| 속성 | 값 |
|------|-----|
| 기준 해상도 | 375×812 (모바일, iPhone 기준) |
| 플랫폼 | iOS / Android / Mobile Web |

### 1-2. 레이아웃 구조

```
┌───────────────────────────────────┐
│ Status Bar (50px)                 │
├───────────────────────────────────┤
│ Top Navigation (44px)             │
│ (← 뒤로가기 or X 닫기 + 페이지 타이틀) │
├───────────────────────────────────┤
│                                   │
│  Content Area (스크롤 가능)         │
│  padding: 0 16px                  │
│                                   │
│  ① Page Heading                  │
│     (타이틀 + 서브텍스트)             │
│                                   │
│  ② Body / Form                   │
│     (컴포넌트 조합)                  │
│                                   │
├───────────────────────────────────┤
│ Sticky Bottom Bar (60px)          │
│ (CTA 버튼)                        │
└───────────────────────────────────┘
```

| 영역 | 속성 |
|------|------|
| Status Bar | 상단 고정, height 50px |
| Top Navigation | 상단 고정, height 44px |
| Content Area | 상하 스크롤 가능, 좌우 padding 16px |
| Page Heading | 타이틀(Bold) + 서브텍스트(Regular), padding-bottom 32px |
| Sticky Bottom Bar | 하단 고정, height 60px, padding 좌우 16px, 상하 8px |

## 4. 간격 & 정렬

### 4-1. Spacing 토큰

| 값 | 용도 |
|----|------|
| 2px, 4px | 텍스트와 보조 텍스트 사이 간격 |
| 6px | 아이콘-텍스트 간격 |
| 8px | 관련 요소 간 최소 간격 |
| 10px, 12px, 16px | 섹션 내부 요소 간격 |
| 16px | 좌우 페이지 마진 (고정) |
| 24px, 30px | 섹션 간 구분 간격 |

### 4-2. 폼 레이아웃

```
┌─────────────────────────────────────┐
│ 레이블 텍스트 (Body16_Regular)        │
│ ↕ 8px                               │
│ [      Input Field      ]           │
│ ↕ 16px                              │
│ 레이블 텍스트                          │
│ ↕ 8px                               │
│ [  Input  ] [  Input  ]  ← 2열      │
└─────────────────────────────────────┘
```

| 속성 | 값 |
|------|-----|
| 레이블 위치 | Input 상단 |
| 레이블-Input 간격 | 8px |
| 필드 간 간격 | 16px |
| 2열 Input 사이 간격 | 8px |
| 좌우 페이지 패딩 | 16px |

|------|------|--------|

**코드 예시**

```tsx
import { BoxButton } from '@bucketplace/design-system'

// Enabled
<BoxButton variant="Primary1" shape="Filled" size="44" state="Enabled" title="다음" />
```

### 5-2. Input Field

**스펙**

| 속성 | 값 |
|------|-----|
| 컴포넌트 | `Input` |
| size | 40 (height 40px) |
| 너비 | 전체 너비 (기본) 또는 비율 분할 |
| 텍스트 | Body16_Regular |
| 플레이스홀더 | Body16_Regular, muted-foreground 색상 |

**변형 케이스**

| 케이스 | 설명 | 예시 |
|--------|------|------|
| 기본 | 단일 입력 | 상세주소 입력 |
| 2열 분할 | 동등 너비 2개 | 성(Last Name) / 이름(First Name) |
| 버튼 조합 | Input + Box Button | 우편번호 + 주소 검색 버튼 |
| 인증번호 | Input + 타이머 + 재요청 버튼 | 인증번호 입력 `01:29` + 인증번호 재요청 |
| 주민등록번호 | 앞자리 Input + 뒷자리 (1자리 노출 + 마스킹 ●●●●●●) | |
| 금액 입력 (AmountInput) | 하단 라인형 Input, 큰 텍스트 | 충전할 금액을 입력해주세요 |

**코드 예시**

```tsx
import { Input } from '@bucketplace/design-system'

// 기본
<Input size={40} placeholder="상세주소 입력" />

// 2열 분할 (각 (100% - 8px) / 2)
<div style={{ display: 'flex', gap: '8px' }}>
  <Input size={40} placeholder="성(Last Name)" />
  <Input size={40} placeholder="이름(First Name)" />
</div>

// 버튼 조합 (Input + BoxButton)
<div style={{ display: 'flex', gap: '8px' }}>
  <Input size={40} placeholder="우편번호" style={{ flex: 1 }} />
  <BoxButton variant="Line" shape="Filled" size="40" title="주소 검색" />
</div>
```

```tsx
/>
```
